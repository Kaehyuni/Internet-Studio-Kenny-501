Domain Name: Semester Project

I met with my classmate Jamie to review possible domains for my semester project. We focused on clarity, professionalism, and alignment with an academic audience. Each idea references my initials and my role at the Richland Learning Commons so the purpose of the site is evident to students and faculty.

My preferred choice is khn-rlc-ar3d-learning.com. KHN is my initials and RLC refers to the Richland Learning Commons where I work as a tutor. The phrase “ar3d learning” communicates the site's focus on augmented and 3D learning. Jamie liked this option because it signals education clearly and supports a professional tone for an academic website.

Reflection: What I Practiced

I practiced applying color with different CSS formats and matched those values to a consistent palette. I used hex codes for direct color references, rgba() when I needed opacity, and hsla() to control hue, saturation, and lightness. Mapping these to my five brand colors helped me understand how each format behaves while keeping the design consistent.

HEX examples (Gunmetal & White)

/* HEX — exact values from the palette */
.color-gunmetal { color: #022B3A; }
.color-white    { color: #FFFFFF; }

RGBA example (Teal with transparency)

/* RGBA — Teal with 90% opacity */
.teal-text { color: rgba(31, 122, 140, 0.9); }

HSLA examples (Columbia Blue & Lavender)

/* HSLA — approximations matching the palette */
.columbia-text { color: hsla(210, 78%, 86%, 1); }  /* #BFDBF7 */
.lavender-text { color: hsla(226, 40%, 92%, 1); }  /* #E1E5F2 */
